Tentang KR RUN 2025

Mengapa KR RUN?

KR RUN tidak hanya sekadar fun run, melainkan juga sebuah perjalanan sejarah yang membawa pelari menyusuri keindahan kultur dan tradisi. Bergabunglah dengan kami untuk merayakan semangat olahraga secara bersama!

Pengalaman Berlari yang Unik

Setiap langkah di KR RUN mengajak Anda untuk menjelajahi tempat-tempat bersejarah dan menikmati keindahan budaya lokal. Ini adalah momen berharga bagi setiap pelari!

Kenali KR RUN

KR RUN adalah ajang yang dirancang untuk semua level pelari. Dari pemula hingga profesional, setiap orang dapat menikmati berlari sambil mengeksplorasi kekayaan budaya. Mari bergabung dalam perjalanan ini!

Umum

  1. KR RUN adalah acara lari yang diselenggarakan oleh Kedaulatan Rakyat di Yogyakarta.
  2. Acara ini bertujuan untuk meningkatkan kesadaran akan pentingnya olahraga dan gaya hidup sehat.
  3. KR RUN terbuka untuk semua warga negara Indonesia yang berusia 17 tahun ke atas.
